一、外网一次性准备

确认外网 Node 版本 必须 24.14.1
外网安装依赖和构建时必须与内网 Node 对齐，避免 node_modules 版本不匹配。
node -v
外网安装依赖并构建
只在外网执行。build 已避开旧版 vue-tsc 的 Node 24 兼容问题，内网不要下载升级。
npm install
npm run build
打包完整目录 含 node_modules
必须包含 node_modules、dist / dist-electron、electron、python、package.json、README.md。

二、内网验收

确认内网 Node 版本 必须 v24.14.1
如果版本不一致，先停下，不要在内网 npm install。
node -v
验证 node_modules 已随包带入
只做 require 检查，不触发下载。
node -e "require('./node_modules/electron-log'); console.log('依赖完整')"
启动开发调试
用于打开主窗口和 RPA 子窗口。
npm run dev
