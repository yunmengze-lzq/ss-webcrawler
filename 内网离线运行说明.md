# 内网离线运行说明

## 推荐使用方式

把整个离线包复制到内网电脑后，解压，双击：

```text
启动-智能体取数工具.bat
```

这个方式不需要在内网执行 `npm install`，也不需要下载 Electron、Playwright 或 Python 包。
启动脚本会优先使用包内自带的 Electron 运行时。

## 包内必须包含

- `node_modules/`：外网准备好的依赖和 Electron 运行时。
- `dist/`：前端构建产物。
- `dist-electron/`：Electron 主进程构建产物。
- `python/`：SQLite 和智能体计算脚本。
- `package.json`：Electron 入口配置。

## 注意事项

- 内网不要执行 `npm install`。
- 内网不要执行 `npx playwright install`。
- 如果 Cookie 过期，在工具中使用“更新Cookie”或重新编辑接口配置。
- 接口返回会先保存为 `raw.json`，再按字段映射生成 `rows.json`，最后按配置写入 Excel 或 SQLite。

## 重新构建

如果在外网电脑改了代码，再执行：

```powershell
node scripts\build-offline.mjs
```

然后重新打包整个目录给内网使用。
